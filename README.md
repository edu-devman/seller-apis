### Размещение товаров фирмы Casio

Последнии модели устройств (часы, калькулыторы и д.р.) фирмы Casio представлены
на сайте ["Группа АВГУСТ"](https://timeworld.ru/products/casio/)

Для получения с последующим размещением и продажей этих моделей через сеть `ozon.ru` необходимо применить команду:

```python.exe seller.py```

В результате в вашем личном кабинете будут обновлены категории товаров фирмы Casio...

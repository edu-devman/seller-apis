# Marketplace worker

## О проекте

Автоматизация процессов обновления и размещения товаров
в маркетплейсах Озон и Яндекс Маркет

## Работа с маркетплейсами

### Озон

Подробнее про работу с [Ozon](https://seller-edu.ozon.ru/)

### Размещение товаров в Озон

Программа ``Seller`` позволяет размещать товары в маркетплейсе Озон.

В настоящий момент ``Seller`` работает только с товарами фирмы Casio,
представленными официльным диллером 
["Группа АВГУСТ"](https://timeworld.ru/products/casio/)

✔️ ``Seller`` позволяет загружать список товаров, с актуальными ценами и размещать
их на сайте `ozon.ru`.

✔️ ``Seller`` обновляет цены и остатки товаров, таким образом на сайте `ozon.ru`
всегда представлена актуальная информация от диллера - `"Группа АВГУСТ"`

🟢 Для запуска ``Seller`` выполните следующию команду:

```python.exe seller.py```

### Ядекс Маркет

Подробнее про работу с [ЯндексМаркет](https://yandex.ru/support/marketplace/ru/start/register?ysclid=mhvutprrd0403413080)

### Размещение товаров в Яндекс Маркет

Программа ``Market`` позволяет размещать товары в маркетплейсе Яндекс Марке.

В настоящий момент ``Market`` работает только с товарами фирмы Casio.

❗ Информация о товарах актуализируется с помощью программы ``Seller``.

✔️ ``Market`` получает список товаров, с актуальными ценами и размещать
их на сайте `Яндекс Марке`.

✔️ ``Market`` обновляет цены и остатки товаров, таким образом на сайте `Яндекс Марке`
всегда представлена актуальная информация от диллера - `"Группа АВГУСТ"`

🟢 Для запуска ``Market`` выполните следующию команду:

```python.exe market.py```
